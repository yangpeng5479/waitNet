# 音频压缩工具 · 用户操作手册

## 1. 软件简介

**音频压缩** 是一款桌面端 MP3 批量压缩工具，内置 FFmpeg。

| 语言 | 应用名称 |
|------|----------|
| 中文 | 音频压缩 |
| English | Audio Compressor |
| العربية | ضاغط الصوت |

主要能力：

- 批量压缩文件夹内所有 MP3
- 支持选择单个 MP3 文件压缩
- 每个文件独立进度条与状态显示
- 压缩前后体积对比（单文件 + 汇总）
- 支持中文 / 英文 / 阿拉伯语（阿语为 RTL 镜像布局）
- 右上角可手动切换语言

## 3. 基本操作流程

### 方式 A：批量压缩整个文件夹

1. **选择源文件夹**（包含 `.mp3` 文件的目录）
2. **选择目标文件夹**（输出目录，不存在会自动创建）
3. 点击 **「扫描 MP3」**，加载任务列表
4. 确认压缩配置（默认已是「推荐配置」）
5. 点击 **「开始压缩」**
6. 在任务列表中查看每个文件的进度与完成状态
7. 全部完成后会播放提示音

### 方式 B：压缩单个 MP3 文件

1. 点击 **「选择单个 MP3 文件」**
2. 选择目标文件夹
3. 点击 **「扫描 MP3」** 或 **「开始压缩」**
4. 仅压缩该文件

## 4. 压缩配置说明

### 4.1 预设模式

| 预设 | 说明 |
|------|------|
| **推荐配置**（默认） | 单声道 + libmp3lame + VBR qscale 8，体积与音质平衡 |
| **高压缩** | 单声道 + 64kbps CBR，体积更小 |
| **高质量** | 立体声 + VBR qscale 4，音质更好 |
| **自定义** | 手动调整声道、比特率、VBR 质量、采样率 |

> 非「自定义」预设时，下方参数不可编辑；切换到「自定义」后可手动调整。

### 4.2 推荐配置等价命令

```bash
ffmpeg -i input.mp3 -ac 1 -codec:a libmp3lame -qscale:a 8 output.mp3
```

## 5. 批处理控制按钮

| 按钮 | 作用 |
|------|------|
| **扫描 MP3** | 扫描源目录（或已选单文件）并加载任务列表 |
| **开始压缩** | 开始批量压缩 |
| **暂停** | 暂停当前压缩（可继续） |
| **继续** | 从暂停处恢复 |
| **停止** | 停止当前批处理 |
| **重试失败项** | 仅重新压缩失败的任务 |
| **清空任务** | 清空任务列表与进度（**不删除**任何文件或文件夹） |

## 6. 任务列表与状态

每个文件一行，包含：

| 列 | 说明 |
|----|------|
| 文件名 | 源 MP3 文件名 |
| 原始大小 | 压缩前体积 |
| 压缩后 | 压缩后体积 |
| 节省 | 节省的体积与百分比 |
| 进度 | 进度条 + 百分比 |
| 状态 | 等待中 / 压缩中 / 完成 / 失败 / 已取消 |
| 备注 | 错误信息或完成提示 |

列宽可用鼠标拖拽调整。

## 7. 语言设置

### 7.1 自动检测

启动时根据系统区域设置自动选择语言：

- 中文系统 → 中文
- 英文系统 → 英文
- 阿拉伯语系统 → 阿拉伯语
- 其他语言 → 默认中文

### 7.2 手动切换

右上角 **语言下拉框** 可随时切换：

- 中文
- English
- العربية（切换后界面为 RTL 镜像布局）

## 8. 特殊规则

### 8.1 源目录与目标目录相同

若源文件夹和目标文件夹选为同一路径，程序会自动将输出改为：

```
源目录/outFile/
```

压缩后的文件保存在 `outFile` 子文件夹中，避免覆盖原文件。

### 8.2 全部完成提示

当本轮所有任务均成功完成时，会播放系统提示音。

### 8.3 输出文件位置

- 默认：输出到所选**目标文件夹**，保持原文件名
- 源=目标时：输出到 `源目录/outFile/`，保持原文件名

## 9. 常见问题

### Q1：压缩后文件反而变大？

源文件可能已是低码率 MP3。可尝试「高压缩」预设，或在自定义模式下提高 qscale 数值。

### Q2：压缩失败怎么办？

1. 检查源/目标路径是否正确
2. 点击「扫描 MP3」刷新任务列表
3. 点击「重试失败项」单独重试失败文件

### Q3：如何只清空任务、不删文件？

点击 **「清空任务」**。该操作仅清空界面任务列表，不会删除磁盘上的任何文件或文件夹。
